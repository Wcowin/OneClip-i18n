# OneClip i18n 资源库

本仓库用于存放 **OneClip** 的多语言翻译资源（i18n）。  
欢迎大家一起完善翻译、修正错误、补充新语言！

---

## 目录说明

> 以下为示例说明，请根据实际结构调整。

- `locales/`
  - `en/`：英文翻译文件
  - `zh-CN/`：简体中文翻译文件
  - `...`：其他语言
- 其他与国际化相关的配置文件

---

## 如何参与贡献

非常欢迎任何形式的贡献，尤其是以下几类：

- **新增语言**：为 OneClip 增加全新的语言支持
- **补充/修正翻译**：完善现有语言中的缺失或不准确翻译
- **术语统一**：统一关键术语的翻译风格与用词
